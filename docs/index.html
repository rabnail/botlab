BEM API Reference

Introduction

Welcome to the BEM api documentation page. Here you can find all information needed to access the api and start building bots for the Bots platform.

In order to gain access to the api, you need to create a bot name and obtain a key for it. Both can be done in the app. Please see the bot creator documentation how to achieve this. You can find it here

The BEM api is REST api that gives your bots access to the Bots platform. It enables you to create Orders and Signals; both are explained here.

URL

When developing/testing your bots, you should use: https://signal.revenyou.io/paper/api/signal/v2
Once your bot comes out of quarantine use: https://signal.revenyou.io/production/api/signal/v2

OpenAPI description

The OpenAPI file can be downloaded here: BEM.yaml.
Use this to explore the BEM API by uploading it in the Swagger editor
